ブラウザ三国志 施術ループ自動候補テスト版

（武将名＋スキルLvリスト出力）

本リポジトリは、ブラウザ三国志の「内政設定ページ」 上で
施術ループが成立する可能性のあるスキル並びを自動解析し、
成立時に 「番号付きの武将名＋スキルLv一覧」 を出力する
研究・検証目的のテスト専用スクリプト です。

※ 本ツールは完成版ではなく、
　施術ループ解析ロジックの深化・検証を目的としています。

特徴・安全性

外部サーバーへの通信は 一切行いません

ブラウザ三国志の ページDOM構造のみをローカル解析 します

アカウント情報・Cookie・通信内容の送信等はありません

Waterfox + Greasemonkey にて動作確認済み

使用方法

ブラウザ三国志の 「内政設定」ページ を開く

画面右側（ゲーム画面外側）に
　「施術ループ候補解析（テスト版）」 ボタンが表示されていることを確認

テストしたいカード構成の拠点を選択した状態でボタンをクリック

結果が以下のいずれかで表示されます

成立時：

1. 武将名　スキル名LV○
2. 武将名　スキル名LV○
...


不成立時：

ループは見つかりませんでした


注意事項（テスト版について）

本ツールは ループ成立を保証するものではありません

最小手数を必ず導出することを保証するものではありません

カード構成・Lv・スキル内容により結果は変動します

将来的にロジック・判定条件は変更される可能性があります

施術ループ解析ロジックの改善に大変役立ちます。
