ブラウザ三国志 施術ループ自動候補テスト版

（武将名＋スキルLvリスト出力）

本リポジトリは、ブラウザ三国志の「内政設定ページ」上で、施術ループが成立する最適なスキル並びを自動解析し、
成立時に “番号付きの武将名＋スキルLvリスト” を出力するシンプルなテスト専用スクリプトです

外部サーバーへの通信は一切行いません
ブラウザ三国志のページ構造をローカルで解析します
WATERFOX + Greasemonkey で動作確認済み

使用方法
ブラウザ三国志の 内政設定 を右クリックで開く
ゲーム画面の右上に　施術ループ候補解析（テスト版）　のボタンが表示されていれば
テスト可能です
テストしたいカードをセットした拠点を選択してボタンを押してください
ループが成立していたら　カード名　と　スキル　のリストが表示されます
不成立の場合は　ループは見つかりませんでした　と表示されます

テスト版でありループの成立を補償するものではありません
最小手数を補償するものではありません
